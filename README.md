# FoxLaby Tools

### Install Tools

```
$ git clone https://github.com/FoxLaby/tools
$ cd tools
$ npm install -g
```

## Create a new angularjs application
Publish the project files into a folder
```
$ foxlaby new ngjs
```
Create a folder named Project
```
$ foxlaby new ngjs <APPNAME>
```
Install packages
```
$ bower install
```
